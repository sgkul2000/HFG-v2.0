A first successful event where many enthusiastic students tried to build their own Arduino based project.

Episode 2

In this episode, participants got their own hardware components which they used to make various fun circuits. Servo motor and LDR were discussed in detail.

Episode 3

In this episode, participents learnt the basics of serial communication protocols such as I2C, SPI and how to interface sensors lik DHT11 and HC-SR04.

In this episode, participents learnt how to read a datasheet and register map inorder to interface MPU-6050. Participants were demonstrated with a mini project on Thrust vector control using servos and IMU sensors.

Episode 5 is a ebook session on creating a weather station using algorithms like nueral network and linear regression one low power microcontrollers like Arduino.
